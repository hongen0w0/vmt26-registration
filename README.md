# VMT26 Landing Page

A Vercel-ready landing page for Velocity maimai Tournament 2026.

The site does not store user information. Registration buttons link directly to the official Google Form.

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy To Vercel

1. Push the repository to GitHub.
2. Import the project in Vercel.
3. Keep the framework preset as Next.js.
4. Deploy.

No database, admin password, or environment variables are required for the landing page.
